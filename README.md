# Cyber-Calculator
C++ program to generate and analyze password strength.
***Project Title: Cyber Calculator
***Description:
Cyber Calculator is a tool designed to generate strong passwords and analyze their strength using information entropy. It applies combinatorial principles to calculate the total search space and estimates the time required for a brute-force attack. This project is to fortify the first line of digital defense.
